🚀 GitHub Guide

✅ What is GitHub?

GitHub is an online platform that allows developers to:

  • Store their code.
  • Share and collaborate with others.
  • Track changes in code using Git.
  • Deploy their projects easily.

✅ Why Do We Need GitHub?

🚩 Problem #1: You lost your code! 😭

If your laptop crashes, your entire code will be gone. But if you push your code to GitHub, it will be safe online.

🚩 Problem #2: Team Collaboration 💻👨‍💻

Multiple developers can work together, merge code easily, and track changes.

✅ How GitHub Works with Git?

Git is a version control system that runs on your computer, while GitHub is an online cloud storage for your Git repositories.

✅ Common GitHub Commands

Command Purpose Example
git init Initialize Git in your project git init
git add . Stage all changes git add .
git commit -m "message" Commit changes git commit -m "Initial Commit"
git push -u origin main Push code to GitHub git push -u origin main

✅ How to Connect Your Project to GitHub (Step-by-Step)

💻 Step 1: Initialize Git

git init

💻 Step 2: Stage Your Changes

git add .

💻 Step 3: Commit Your Code

git commit -m "First Commit"

💻 Step 4: Create a GitHub Repository

Go to GitHub and create a new repository.

💻 Step 5: Connect GitHub to Your Project

git remote add origin https://github.com/username/repo.git

💻 Step 6: Push Your Code to GitHub

Change the Branch to main

git branch -M main

Puch command to githum

git push -u origin main
